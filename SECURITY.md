# Security Policy

Betta-Morpho is a security research and scanning platform.

Please report vulnerabilities responsibly and privately.

## Supported Scope

This repository currently supports private security reports for:

- code execution vulnerabilities
- sensitive data exposure
- authentication or authorization bypass in project services or scripts
- packaging or supply-chain issues introduced by this repository

This repository does not accept public issue posts for unpatched security
vulnerabilities.

## How To Report

Send a private report to the maintainer contact listed in the repository
metadata and legal files.

Include:

- affected file or command path
- clear reproduction steps
- impact assessment
- environment details
- proof-of-concept only as needed

## Safe Reporting Rules

- do not publish exploit details before coordinated review
- do not test outside systems you do not own or control
- do not attach unrelated third-party data
- keep proofs minimal and contained

## Response Intent

The project aims to:

- acknowledge clear reports
- reproduce the issue
- patch or mitigate where feasible
- document user-facing security impact when relevant

No guaranteed SLA is promised.

## Operational Note

If you discover a risk involving scan artifacts, packet captures, or runtime
telemetry, assume those files may contain sensitive infrastructure information
and handle them accordingly.
