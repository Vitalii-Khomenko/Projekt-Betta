- you may refer to the project by name when describing compatibility, origin, or factual use

- you may not imply endorsement, sponsorship, or official affiliation by using the project name or branding in a misleading way
- you may not reuse logos, visual branding, or product identity as if they were your own product marks

Acceptable examples:

- "Forked from Betta-Morpho"
- "Compatible with Betta-Morpho artifacts"
- "Derived from the Betta-Morpho scanner codebase"

Not acceptable examples without separate permission:

- shipping a fork under the same primary product name in a way that confuses users
- using Betta-Morpho logos or branding in commercial materials as if officially endorsed
- presenting modified versions as the official Betta-Morpho release

If you want to use the project name or branding beyond nominative reference,
obtain separate written permission from the project author.
